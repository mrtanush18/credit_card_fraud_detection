# How to Run Locally
Clone the repository:
```
git clone https://github.com/mrtanush18/credit_card_fraud_detection.git
cd credit_card_fraud_detection
```
Install dependencies:
```
pip install -r requirements.txt
```
Run the app:
```
streamlit run app.py
```
